# Go-Time

Web application for the Go-Time platform. Built in React.

Run locally:

```powershell
npm install --legacy-peer-deps
npm run dev
```

Open the URL printed by `vite` (usually http://localhost:5173).

Build:

```powershell
npm install --legacy-peer-deps
npm run build
```

Run production version:

```powershell
npm start
```
